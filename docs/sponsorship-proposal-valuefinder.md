# Prism Insight × ValueFinder 스폰서십 제안서

> **제안 요약**: 독립 오픈소스 리서치 트래커 Prism Insight가 ValueFinder와 파트너십을 맺어, 리포트의 시장 영향력을 데이터로 실증하고 투자자 커뮤니티를 공동으로 성장시킵니다.

---

## 1. Prism Insight 소개

**Prism Insight**는 국내 독립 리서치 보고서의 시장 반응을 실시간으로 추적·분석하는 오픈소스 프로젝트입니다.

- GitHub 공개 오픈소스, 누구나 기여 가능
- 리포트 발간일 기준 주가 추적 (발간가 → 최고가 → 현재가)
- 인사이트 대시보드: 리포트 경과 기간별 성과, 테마별 시장 반응, 빈티지(연도별) 분석 등
- Vercel 기반 정적 배포, 매일 자동 데이터 업데이트

현재 **ValueFinder** 리포트를 1차 추적 대상으로 운영 중이며, 2021년부터 현재까지 **263건+** 리포트의 시장 반응 데이터를 보유합니다.

---

## 2. 왜 ValueFinder인가

ValueFinder는 국내 중소형주 리서치 시장에서 독보적인 포지션을 가지고 있습니다.

| 지표 | 수치 |
|------|------|
| 추적 리포트 수 | 263건+ (2021~현재) |
| 리포트 발간 추이 | 월 2~4건(2021) → 월 8~11건(2026), 3배+ 성장 |
| 발간 후 평균 최고 상승률 | 중앙값 기준 연도별 20~77% |
| 커버 테마 | AI/반도체, 2차전지, 바이오, 방산, 로봇 등 |

---

## 3. ValueFinder가 얻는 것

### 3-1. B2B 영업 무기: 독립 데이터로 검증된 ROI

고객사(피칭 대상 기업)에게 가장 설득력 있는 증거:

> *"ValueFinder 리포트 발간 후 평균 X일 이내에 Y% 상승이 독립 트래커에서 외부 검증됩니다"*

자체 제작 자료가 아닌 **외부 오픈소스**가 이를 측정한다는 점이 신뢰도를 높입니다.

### 3-2. 투자자 커뮤니티 확보

Prism Insight 이용자 = ValueFinder 리포트를 데이터로 추적하는 적극적 투자자층

- 리포트 발간 알림 → 트래커 조회 → 리포트 직접 방문의 자연스러운 퍼널
- 투자자가 "어떤 타이밍에 활용해야 하나"를 스스로 분석 → 리포트 충성도 상승

### 3-3. 브랜드 신뢰도

독립적인 성과 추적을 허용하는 리서치 회사 = **데이터에 자신 있는 회사**라는 포지셔닝

## 4. 스폰서십 옵션

### Option A — 로고 스폰서 (기본)

| 항목 | 내용 |
|------|------|
| ValueFinder 제공 | 공식 로고, 리포트 목록 RSS 또는 API |
| Prism Insight 제공 | 대시보드 상단 "Powered by ValueFinder" 배너, 각 리포트에 원문 링크 |
| 금액 | 무상 (상호 홍보) |

### Option B — 공동 마케팅

| 항목 | 내용 |
|------|------|
| ValueFinder 제공 | 뉴스레터 / SNS에 Prism Insight 소개, 데이터 제공 협조 |
| Prism Insight 제공 | 전용 ValueFinder 성과 리포트 월 1회 발행, 공동 보도자료 |
| 금액 | 월 협찬금 협의 또는 무상 교환 |

### Option C — 데이터 파트너십 (프리미엄)

| 항목 | 내용 |
|------|------|
| ValueFinder 제공 | 리포트 발간 사전 알림 API, 애널리스트 코멘트 |
| Prism Insight 제공 | ValueFinder 전용 B2B 성과 대시보드 (영업 자료용 PDF 생성 포함) |
| 금액 | 수익 분배 또는 고정 계약 협의 |

## 5. 제안하는 첫 단계

1. **킥오프 미팅** (30분): 현재 트래커 데모 시연, 데이터 현황 공유
2. **Option A 시범 운영** (1개월): 로고 배치 + 성과 측정
3. **확장 여부 결정**: 트래픽·전환율 데이터 기반으로 Option B/C 논의
